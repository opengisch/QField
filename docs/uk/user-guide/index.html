Посібник користувача QField¶

Основні операції¶

Зміна режимів¶

Режими перемикаються за допомогою Menu button ‣ mode

Відкриття проекта¶

Ідентифікація об’єктів¶

Натискання (тап) на об’єкт виконає ідентифікацію. Настискання кнопки “Назад” закриє вікно ідентифікації.

Виключення під час ідентифікації шарів¶

Часто ідентифікація всіх шарів небажана. Деякі шари виконують роль базових та їх атрибути нас не цікавлять.

Ви можете налаштувати цей список шарів у QGIS в Project ‣ Project Properties ‣ Identify Layers та відключити ідентифікацію для базових шарів.

GPS¶

Коротке натискання на кнопку GPS вмикає GPS та центрує на поточному місцезнаходженні, як тільки з’явиться інформація про положення.

Довге натискання кнопки GPS відображає меню позиціонування.

У меню позиціонування ви можете активувати відображення положення, тобто відображення поточних координат, перепроектованих у систему координат проекта, та інформації про їх точність.

Примітка

Якщо замість інформації у системі координат проекта ви бачите інформацію у WGS 84 (широта/довгота) , можливо ви ще впіймали сигнал.

Використання зовнішнього GNSS-приймача¶

Існує можливість передавати інформацію про місцезнаходження з іншого додатку до QField. Наприклад, з додатку Android NTRIP Client.

Для цього Вам потрібно увімкнути отримання інформації про місцезнаходження на Вашому пристрої Android.

Оцифровка¶

Щоб почати оцифровку нових об’єктів перейдіть у режим оцифровки, як описано в розділі Зміна режимів.

Поряд з кнопкою меню з’явиться новий випадаючий список з переліком шарів, доступних для оцифровки.

Точки¶

Помістіть курсор у центрі екрана в бажане місце та натисніть на олівець у правому нижньому кутку екрана, щоб підтвердити створення нового точкового об’єкта.

Лінії та полігони¶

Помістіть курсор у центрі екрана в бажане місце початку лінії та натисніть на олівець у правому нижньому кутку екрана. Продовжуйте додавати чи видаляти точки поки лінія чи полігон не набудуть бажаного вигляду, а потім натисніть зберегти.

Заповнення атрибутів¶

After digitizing a geometry, the attribute form will be displayed if it is not suppressed and the user will be asked to enter the attributes for the new feature. The form which appears allows entering attribute values for the new feature. The checkboxes at the right of every attribute allow for remembering each attribute individually.

Видалення об’єктів¶

Видалення об’єктів можливе лише в режимі оцифровки.

Щоб видалити об’єкт, виконайте ідентифікацію та клацніть по іконці сміттєвої корзини.
