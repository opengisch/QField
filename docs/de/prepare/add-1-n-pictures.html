Eine Reihe von Bildern zu einem Feature hinzufügen¶

Es ist möglich auch mehr als ein Bild zu einem Feature hinzuzufügen. Im Folgenden ein Beispiel, wie das umgesetzt werden kann.

Zwei Tabellen erzeugen¶

Wir benötigen zwei Tabellen. Eine Tabelle, in welcher die Features gespeichert werden und eine, in der Pfade zu den Bildern abgelegt werden.

Apiary¶

  • id (UUID)
  • geometry
  • …

Apiary_pictures¶

  • id
  • apiary_id (UUID)
  • path (TEXT)
  • …

Relations¶

Erstelle eine Beziehung mit:

  • apiary Referenced layer
  • id Referenced field
  • apiary_picture Referencing layer
  • apiary_id Referencing field
  • strength Composition

Widgets¶

apiary¶

Setze den Standardwert des Feldes id auf uuid(). Es ist nicht nötig, diesen im Formular zu zeigen.

Setze das Beziehungswidget auf Viele zu einem und füge die Beziehung zum Formular hinzu.

picture¶

Setze den Widgettypen des Feldes path auf Attachment und füge es zum Formular hinzu
