Installation¶

Android¶

QField can be used with any version newer than Android 5, if you are looking to get a new device, we suggest going for at least Android 9.

Get a pre-built apk¶

We also offer a pre-built apk for you to download if you are not able or do not want to use GooglePlay. You can get the qfield apk from the release page.

Windows¶

Windows support is currently in beta status. Download the latest development builds and leave us a note how it works.

iOS¶

An iOS version is in the making. You can sign up for beta testing to get access earlier than everyone else.
