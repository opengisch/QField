Synchronisation¶

La synchronisation des projets avec des appareils mobiles peut être réalisé de différentes manières. Le plugin QFieldSync permet de préparer le projet QGIS regroupé avec les données. Ces fichiers peuvent ensuite être l’ouverture sur vos appareils mobiles. Pour faciliter l’intégration et la synchronisation des projets QGIS entre différents appareils mobiles, nous vous incitons à utiliser QFieldCloud.
