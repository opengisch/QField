Formats de données supportés¶

QField supporte une grande quantité de formats via le provider QGIS et GDAL. Cette page offre une liste non exhaustive des formats supportés.

Données raster¶

Les données raster peuvent vite devenir très lourdes si l’on travaille avec du tiff sans compression, jusqu’à plusieurs Go de données. Spécialement sur des appareils mobiles, ce n’est pas efficace.

Utiliser GeoPackage¶

Nous recommandons d’utiliser le format GeoPackage pour travailler avec des données raster. Les commandes suivantes permettent de convertir un fichier nommé raster.tif en un fichier raster.gpkg avec pyramides. Remplacer EPSG:21781 (CH1903 LV03) avec votre SCR.

gdal_translate --config OGR_SQLITE_SYNCHRONOUS OFF -co  APPEND_SUBDATASET=YES -co TILE_FORMAT=PNG_JPEG -a_srs EPSG:21781 -of GPKG raster.tif raster.gpkg
gdaladdo --config OGR_SQLITE_SYNCHRONOUS OFF -r AVERAGE raster.gpkg 2 4 8 16 32 64 128 256
