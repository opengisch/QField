Android : note spécial d’accès aux fichiers¶

Symboles SVG personnalisés et configuration¶

Le chemin aux dossiers partagés de l’appareil peuvent être consulté dans la boîte de dialogue “A propos”.

Les symboles personnalisés doivent être copiés à l’aide d’un gestionnaire de fichiers.

Carte SD externe¶

Pour avoir accès en écriture aux projets qui sont sur une carte SD externe, il faut les mettre dans le dossier suivant :

/Android/data/ch.opengis.qfield/files
