Interagir avec la carte¶

Légende de carte¶

Ouvrir le menu et dérouler la liste des couches pour afficher la légende de la carte.

Afficher ou non les couches.¶

Menu principal ‣ Appui prolongé sur la couche ‣ Cocher ou décocher afficher sur la carte

Sélectionner des entités¶

Cliquer sur une entité de la carte pour la sélectionner. Si d’autres entités sont proches de cette dernière (soit parce qu’il y a plusieurs entités très proches les unes des autres ou parce que plusieurs couches se chevauchent), elles seront toutes sélectionnées et listées dans le volet à droite de l’écran.

Cliquer sur l’une d’entre elles dans la liste pour accéder à ses attributs.

  1. Utiliser les flèches pour faire défiler les entités sélectionnées.
  2. Cliquer sur le bouton centrer pour centrer la carte sur l’entité sélectionnée.
  3. Cliquer sur le bouton éditer pour modifier les attributs de l’entité sélectionnée.
  4. Faites défiler les sous-menus pour accéder à tous les attributs.

Exceptions couches interrogeables¶

Il n’est souvent pas nécessaire de pouvoir interroger chacune des couches du projet. Certaines couches sont uniquement utiles comme fond de plan et leurs attributs n’ont pas d’intérêt.

Il est possible de gérer la liste des couches interrogeables dans QGIS desktop Projet ‣ Propriétés du projet ‣ Identifier les couches . Décocher à cet endroit les couches qui ne doivent pas être interrogeables.

Outil de mesure¶

Ouvrez le Menu Principal –> Paramètres –> Outil de mesure
