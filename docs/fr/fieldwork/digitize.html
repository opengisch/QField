Collecter les entités¶

Pour commencer à digitlaiser de nouveaux objets, activer le Mode Edition dans le menu principal (bouton en forme de crayon) et sélectionner la couche dans laquelle vous voulez ajouter un nouvel objet (la couche sera mise en surbrillance en vert).

Points¶

Déplacer la carte de sorte que la position désirée se trouve sous le réticule situé au centre de l’écran et cliquer sur sur le bouton + dans le coin inférieur droit pour confirmer la création de cette nouvelle entité.

Si vous avez activé la localisation, vous pouvez utiliser le bouton Centrer pour forcer le centrage de votre position sous le réticule.

Lignes et polygones¶

Déplacer la carte de sorte que le point de départ désiré de la ligne ou du polygone se trouve sous le réticule situé au centre de l’écran. Cliquer ensuite sur le bouton (+) dans le coin inférieur droit pour ajouter le premier nœud.

Procéder de la même manière et répéter l’opération pour former votre ligne ou votre polygone en cliquant sur le bouton (+) chaque fois que vous souhaitez ajouter un nouveau nœud.

Cliquer sur le bouton (-) pour supprimer le dernier nœud ajouté.

Cliquer sur le bouton (X) pour annuler complètement la création de l’entité.

Quand vous avez ajouté au moins 2 nœuds pour une ligne ou 3 nœuds pour un polygone, un bouton Sauvegarder apparaît. Cliquer dessus pour terminer votre dessin et enregistrer l’entité.

Formulaire d’attributs¶

Après avoir tracé la géométrie, le formulaire d’attributs s’affichera s’il n’est pas supprimé et l’utilisateur pourra entrer les attributs du nouvel objet.

Garder en mémoire les valeurs de l’attribut.¶

Pour une collection rapide de jeux de données plus ou moins homogènes il est important de ne pas avoir à rentrer les mêmes valeurs encore et toujours. La case à cocher à droite de chaque attribut permet de garder en mémoire la valeur entrée. Lors de l’ajout d’un nouvel objet dans la même couche, cette valeur sera automatiquement assignée à l’attribut du nouvel objet.

Supprimer des entités¶

La suppression d’entités est uniquement possible en mode Édition.

Pour supprimer une entité, identifiez-la d’abord par une brève pression. Ensuite, sélectionnez-la par une pression longue dans la liste d’identification, sélectionnez d’autres caractéristiques à supprimer si vous le souhaitez et cliquez sur le bouton « Corbeille » pour confirmer la suppression.

Edition des entités¶

Pour modifier, appuyez sur le bouton Menu -> crayon.

Vous pouvez modifier un élément existant en déplaçant, ajoutant ou supprimant le sommet et en le scindant également.

Outil de Découpage¶

Il est possible de découper une géométrie existante en traçant une ligne à l’endroit où elle doit être divisée.

Outil Vertex¶

Il est possible de déplacer et de supprimer des sommets de géométries et d’en ajouter de nouveaux.

Freehand digitizing¶

The freehand digitizing mode allows users to “draw” lines and polygons using their stylus/pen or mouse. The mode is available for line/polygon feature additions as well as the geometry editor’s rings tool.

The freehand digitizing mode is activated through a new toolbar button which appears when QField is set to editing mode and a line or polygon vector is selected.

Topological Editing¶

If topological editing is activated on the project QGIS before exporting for QField, shared nodes of neighbouring geometries are modified together when moving / deleting vertices and additional vertices are added to segments when a new node is added from neightbouring geometries.

Multi editing¶

You can make a multi-selection and edit all the features at the same time. In some cases, it can be very efficient.
