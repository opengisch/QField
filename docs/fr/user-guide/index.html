Guide d’utilisateur QField¶

Workflow basic¶

Changement du mode¶

Le mode peut être changé via Menu button ‣ mode

Ouvrir un projet¶

Interroger des objets¶

Effectuer un “long press”, c’est à dire maintenir le doigt sur l’écran pendant quelques secondes, pour interroger les objets sur la carte. Appuyer sur le bouton retour de l’appareil pour fermer la fenêtre des résultats.

Couche identifiables¶

Il n’est souvent pas nécessaire de pouvoir interroger chacune des couches du projet. Certaines couches sont uniquement utiles comme fond de plan et leurs attributs n’ont pas d’intérêt.

Il est possible de gérer la liste des couches interrogeables dans QGIS desktop Projet ‣ Propriétés du projet ‣ Identifier les couches . Décocher à cet endroit les couches qui ne doivent pas être interrogeables.

GPS¶

Un touché court sur le bouton du GPS l’active et centre la carte sur la localisation actuelle, dès que les données de géoloclisation sont disponibles.

Un touché long sur le bouton du GPS affiche le menu de géolocalisation.

Dans le menu de géolocalisation, l’affichage des données de position peut être activé. Les coordonnées affichées sont reprojetées dans le SCR du projet.

Note

Si les informations sont affichées en WGS84, c’est probablement car aucun signal n’est disponible.

Digitaliser¶

Pour commencer à digitaliser de nouveaux objets `Switch modes`_ to digitizing.

Une nouvelle liste apparaîtra près du bouton du menu. Elle contiendra les couches disponibles pour la numérisation. Pour le moment (v 0.7), QField ne supporte que les points et les lignes.

Points¶

Déplacer la carte jusqu’à ce que la croix du centre de l’écran soit sur la position désirée, puis cliquer sur le crayon situé en-bas à droite de l’écran pour confirmer la création d’un nouveau point.

Lignes¶

Déplacer la carte jusqu’à ce que la croix du centre de l’écran se trouve au début de la ligne, puis cliquer sur le crayon situé en-bas à droite de l’écran. Procéder à nouveau de cette manière pour ajouter ou supprimer des points jusqu’à ce que la ligne soit terminée, puis cliquer sur enregistrer.

Formulaire d’attributs¶

Après avoir numérisé une géométrie, le formulaire d’attributs s’ouvre et l’utilisateur doit saisir les données pour le nouvel objet. Les cases à cocher à droite de chaque attribut permettent d’enregistrer des valeurs par défaut.

Supprimer des objets¶

Deleting features is only possible in digitize mode.

Pour supprimer un objet, l’interroger puis appuyer sur l’icône poubelle.
