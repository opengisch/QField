Guide d’utilisateur QField¶

Flux de travail de base¶

Changement du mode¶

Le mode peut être changé via Menu button ‣ mode

Ouvrir un projet¶

Interroger des objets¶

Une pression sur un objet l’identifiera. Pour clore la fenêtre d’identification, presser le bouton “Retour”.

Exceptions couches interrogeables¶

Il n’est souvent pas nécessaire de pouvoir interroger chacune des couches du projet. Certaines couches sont uniquement utiles comme fond de plan et leurs attributs n’ont pas d’intérêt.

Il est possible de gérer la liste des couches interrogeables dans QGIS desktop Projet ‣ Propriétés du projet ‣ Identifier les couches . Décocher à cet endroit les couches qui ne doivent pas être interrogeables.

GPS¶

Une courte pression sur le bouton du GPS l’active et centre la carte sur la localisation actuelle, dès que les données de géolocalisation sont disponibles.

Une longue pression sur le bouton du GPS affiche le menu de géolocalisation.

Dans le menu de géolocalisation, l’affichage des données de position peut être activé. Les coordonnées affichées sont reprojetées dans le SCR du projet.

Note

Si les informations sont affichées en WGS84 au lieu de votre propre SCR, c’est probablement parce qu’aucun signal n’est disponible.

Utilisation d’un récepteur GNSS externe¶

Il est possible de générer un emplacement fictif grâce à une application android tierce dans qfield. Plusieurs options sont possibles pour cela, l’une d’elles est l’application Android NTRIP Client.

To use this you have to enable mock locations on your Android device.

Numériser¶

Pour commencer à numériser de nouveaux objets `Switch modes`_ to digitizing.

Une nouvelle boîte de dialogue apparaîtra près du bouton Menu pour lister les couches disponibles pour numérisation.

Points¶

Déplacer la carte jusqu’à ce que la croix du centre de l’écran soit sur la position désirée, puis cliquer sur le crayon situé en-bas à droite de l’écran pour confirmer la création d’un nouveau point.

Lignes et polygones¶

Positionner le viseur du centre de l’écran vers le point de démarrage souhaité de la ligne et cliquer sur le crayon en bas à droite de l’écran. Continuer en ajoutant ou supprimant des points jusqu’à ce que la ligne ou le polygone soit terminé puis cliquer sur “Enregistrer”.

Formulaire d’attributs¶

Après avoir numérisé une géométrie, le formulaire d’attributs s’ouvre et l’utilisateur peut saisir les données pour le nouvel objet. Les cases à cocher à droite de chaque attribut permettent d’enregistrer des valeurs par défaut pour chacun d’entre eux.

Supprimer des objets¶

La suppression d’objets est uniquement possible en mode édition.

Pour supprimer un objet, l’interroger puis appuyer sur l’icône poubelle.
