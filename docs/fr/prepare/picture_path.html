Chemin vers l’image configurable¶

QFieldSync donne la possibilité de configurer le chemin des images jointes.

  1. Aller dans QFieldSync –> Configuration du projet
  2. Selectionner la table « Nomination de la photo »
  3. Choisir la couche, le champ et configurer l’expression

Utiliser les expressions pour configurer le chemin vers les fichiers joints. Par défaut, les images sont sauvegardées dans le dossier « DCIM » avec comme nom la date et heure de la capture.

Note

Cette fonctionalité nécessite la version 3.14 de QGIS ou plus.
