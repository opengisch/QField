Autoriser le masquage des nœuds de légende¶

Vous devez d’abord vérifier si l’extension Invisible layers and groups est installée. Si ce n’est pas le cas, vous pouvez l’installer en allant dans Extensions > Installer/Gérer les extensions > Rechercher “Invisible layers and groups” > cliquer sur “Installer l’extension”.

Une fois l’extension installée, vous verrez dans l’interface de QGIS ces boutons :

Une fois le processus d’installation terminé, il est possible de masquer les nœuds de la légende de notre projet.

  1. Choisissez les couches ou groupes que vous souhaitez masquer.
  2. Lorsque vous avez terminé la sélection des couches/groupes, cliquez sur Rendre les couches et groupes sélectionnés invisibles. Vous remarquerez que les couches et groupes sélectionnés ont disparu du panneau des couches. Cependant, ils sont toujours disponibles dans le projet pour la visualisation et les jointures.
  1. Enregistrez votre projet, vous pouvez utiliser la section plug-in QFieldSync pour enregistrer votre projet, ou enregistrer votre géopackage et votre projet dans le même dossier.
  2. Copier les fichiers générés sur votre appareil.
  3. Ouvrez QField dans votre appareil et vous verrez la configuration des nœuds de légende cachés.

Remarque: Si vous souhaitez voir à nouveau les couches/groupes masqués, cliquez sur Afficher les couches et groupes invisibles.
