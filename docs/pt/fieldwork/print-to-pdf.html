Imprimir em PDF¶

É possível exportar os mapas e outros itens para um documento PDF.

If the project has at least one print layout defined, printing is available over the menu.

Após selecionar o layout de impressão, o documento PDF é criado e armazenado no diretório de documentos do dispositivo.

Ao escolher o aplicativo leitor de PDF, o documento pode ser aberto diretamente.

Feito.

Nota

O diretório de documentos do dispositivo é USER>/<APPNAME>/Documents. Para QField geralmente é um caminho como ////storage/emulated/0/Documents/QField
