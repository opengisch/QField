GNSS¶

Um toque breve no botão GNSS irá ligar o GNSS e centrar à sua localização atual assim que a informação de posicionamento estiver disponíveis.

Ative o modo de edição e toque no botão de alvo, a cruz no centro indica que está usar posicionamento GNSS.

Um toque longo no botão GNSS irá mostrar o menu de posicionamento.

Dentro do menu de posicionamento, você pode ativar o ecrã de posicionamento, que mostrará as coordenadas actuais projectadas no SRC do projecto, juntamente com informações de precisão.

Note

Se você vir informações lat / lon do WGS84 em vez de informações no SRC do seu projecto, provavelmente ainda não tem sinal.

Usando um Receptor externo GNSS¶

O QField suporta a ligação a antenas GNSS via bluetooth.

Nas configurações -> posicionamento, os dispositivos bluetooth emparelhados podem ser pesquisados e escolhidos como fonte de posicionamento.

Garanta que não tem outra aplicação com fornecedores de localização simulada a usar a antena bluetooth.

Localização simulada¶

É possível fornecer um local simulado numa aplicação Android separado para o QField. Existem várias opções para isso, uma delas é o Android NTRIP Client <https://play.google.com/store/apps/details?id=com.lefebure.ntripclient> _.

Para usar isto tem de activar as localizações simuladas no seu dispositivo Android.

Para mais informação de como configurar medições de alta precisão e adicionar metadados indicativos de qualidade para recolher elementos, veja a documentação GNSS correspondente GNSS documentation.
