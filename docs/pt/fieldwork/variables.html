Variáveis Globais¶

As variáveis são usadas no QField e no QGIS para definir valores dinamicamente com base no arquivo de projecto actual, na camada ou no sistema. Estes podem ser usados para estilo, definição de valores padrão para campos e muito mais. Clique em Guardar.

Para configurar uma variável específica do dispositivo, abra as configurações e clique no separador «Variáveis Globais».

Configuração de duas novas variáveis globais além das variáveis do sistema predefinidas.

Na página de variáveis, você pode ver variáveis predefinidas. Para adicionar novas variáveis, clique na célula vazia na parte inferior e escreva o conteúdo.
