Interact with the map¶

图例¶

打开**主菜单**，展开图层列表以显示地图图例。

Turning on and off layers¶

Main menu ‣ long press on the layer ‣ check or uncheck show on map canvas

识别要素¶

在地图上点击某要素可以识别它。 如果点击的位置上有多个要素（要么因为有多个要素确实彼此离的很近，要么因为地图中叠加有多个图层），它们都将显示在屏幕右侧的菜单列表中。

点击列表中的某个要素以访问其属性。

  1. 点击**左右箭头**滚动显示识别出的所有要素。
  2. 点击**居中按钮**使得选中的要素在地图上居中显示。
  3. 点击**编辑按钮**可以编辑选中要素的属性。
  4. Scroll through the sub-menus to access all the attributes.

识别图层时的例外情况¶

通常情况下，不需要对所有的图层都进行查询。有些图层仅仅作为底图显示，这些图层的属性也不是关注的重点。

可以在QGIS桌面中管理图层列表，通过`项目 ‣ 项目属性 ‣ 识别图层`找到图层，取消其底图图层选项即可。
