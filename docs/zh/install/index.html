安装¶

安卓¶

QField适用于android5更高版本，我们建议安卓系统版本高于android7。

获取预建的apk¶

如果您不能或不想使用GooglePlay，我们还提供了一个已编译apk供您下载。您可以从release页面获得该apk。

Windows¶

QField for windows测试版可从以下地址获取： https://download.opengis.ch/qfield/ci-builds/win/

iOS¶

iOS版本正在制作中。
