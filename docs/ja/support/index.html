サポートと開発¶

商業サポート¶

QFieldの開発元として, OPENGIS.ch は トレーニングパッケージ と個別の商業 サポート契約 をサービス品質保証契約に基づいて提供することができます。

開発とアプリのカスタマイズ¶

まず実装されていない素晴らしい機能のアイディアをたくさんお持ちかもしれません。 遠慮なく こちら からあなたのアイデアをQFieldで実現する最適なアプローチについて、もしくはあなた自身でのアプリのカスタマイズ方法についてご相談いただければと思います。

コミュニティによるサポート¶

gis.stackexchange にてご質問お待ちしています。

メーリングリスト もご使用いただけます。

支援¶

これらの組織はQFieldを現在の形にするためのご支援をしてくださりました。ご支援に大変感謝いたしております。

クレジット¶

QFieldはOPENGIS.chによって開発されています。OPENGIS.chはQFieldやQGISを始めとするオープンソースソフトウェアに対するコンサルティング、開発、トレーニングを行っています。

APIの安定性¶

QFieldはまだAPIが安定化していません。

ライセンス¶

QFieldは、GNU Public License（GPL）バージョン2以降でリリースされています。 このライセンスの下でQFieldを開発するということは、ソースコードをチェックおよび変更できることを意味し、無償かつ自由に変更可能なQGISベースのフィールドデータコレクションアプリに常にアクセスできることを保証します。

Github でソースコードを閲覧いただけます。
