QFieldユーザーガイド¶

プロジェクトを開く¶

プロジェクトを開くには、 メニューボタン –> オプションボタン –> プロジェクトを開く のようにタップしてください。

開きたいプロジェクトを選択するか、 最近のプロジェクト や お気に入りのフォルダ ショートカットを使用してください。

地図の凡例¶

Open the Main menu and expend the layers list to display the legend of the map.

属性情報の表示¶

地図上のフィーチャをタップすると属性情報を表示できます。もし複数のフィーチャをタップした場合（複数のフィーチャがたがいに近接している場合や複数のレイヤーがオーバーラップしている場合）は、それらすべてのフィーチャが画面の右側のメニューにリストアップされます。

リストアップされたフィーチャの一つをタップして属性情報を表示できます。

  1. 矢印 をタップして全てのフィーチャの属性情報をスクロールできます。

  2. 中央表示ボタン をタップして選択しているフィーチャを中央に表示できます。

  3. 編集ボタン をタップして選択しているフィーチャの属性情報を変更できます。

  4. サブメニュー をスクロールして全ての属性情報を切替表示できます。

特定レイヤーを属性情報表示から除外する¶

全てのレイヤーが属性情報を表示する必要があるとは限りません。例えばベースマップとして表示されているレイヤーなどは属性情報の表示は不要です。

QGISデスクトップの プロジェクト ‣ プロパティ ‣ データソース ‣ 情報表示可能 からベースマップのレイヤーのチェックを外すことで、情報表示対象レイヤーリストを管理できます。

検索¶

右上の 検索ボタン をタップするとレイヤを検索できます。

  1. 検索結果の 名前 をタップするとそのフィーチャを表示できます（検索結果のフィーチャは黄色でハイライトされます）。

  2. 属性ボタン をタップして属性情報を表示することもできます。

マップテーマの切替え¶

もしQGISでこのプロジェクトに対してマップテーマを設定していた場合、 メインメニュー 上の一番上のドロップダウンリストからマップテーマを切り替えることができます。

GPS¶

GPSボタンを短くタップするとGPS機能を有効できます。現在地情報を取得次第、現在地の位置を表示します。

GPSボタンをロングタップするとGPSメニューを表示できます。

GPSメニューの中でプロジェクトの座標系に変換した現在地の座標情報の表示切替などの設定ができます。

注釈

プロジェクトの座標系ではなくWGS84の経度/緯度が表示される場合は、まだGNSS信号を受信できていない可能性があります。

外付けのGNSS受信機を使用する¶

別のAndroidアプリを用いて疑似ロケーションをQFieldに供給することも可能です。いくつかの方法がありますが、その一つとして Android NTRIP Client を活用することもできます。

これを行うには アンドロイド端末で疑似ロケーションを有効 にする必要があります。

デジタイジング¶

新しいフィーチャのデジタイジングをするには、メインメニュー（鉛筆ボタン）から 編集モード を有効にし、新しくフィーチャを追加したいレイヤを選択してください（選択したレイヤーはグレイでハイライトされます）。

ポイント¶

画面の中央にある十字線を目的の場所に移動し、画面の右下にある +ボタン をクリックして、新しいポイントフィーチャの作成を確認します。

GPSを有効にしている場合は、 中央に維持 ボタンを使用して、十字線を現在地の中央に自動的に移動できます。

ラインとポリゴン¶

画面の中央にある十字線を目的のラインまたはポリゴンの開始点に移動し、画面の右下にある プラス（+）ボタン をクリックして最初のノードを追加します。

新しいノードごとに プラス（+）ボタン をクリックして、ポイントを追加してラインまたはポリゴンを形成します。

Click the Minus (-) button to remove the last added node.

Click the Close (X) button to cancel the feature creation completely.

少なくとも2つのノードをラインに追加するか、3つのノードをポリゴンに追加すると、 保存ボタン が表示されます。 クリックしてスケッチを完成させます。

属性情報フォーム¶

図形を作成した後、属性フォームが表示され、ユーザーは新しいフィーチャの属性を入力するよう求められます。 表示されるフォームでは、新しいフィーチャの属性値を入力できます。 すべての属性の右側にあるチェックボックスを使用すると、各属性をデフォルト値として記憶できます。 次回同じレイヤーにフィーチャを追加するとき、これらの属性は最初から入力されています。

フィーチャの削除¶

フィーチャの削除は編集モード時のみ利用可能です。

フィーチャを削除するには、まず属性情報を表示し、ゴミ箱アイコンをタップします。
