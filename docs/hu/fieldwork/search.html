Keresés¶

Koppintson a Keresés gombra jobb felső sarokban az elemek kereséséhez az aktuális projektben.

  1. Koppintson a névre az eredményben, hogy eltoljuk a térképet az elemre (az elem sárgával kiemelten jelenik meg).
  2. Koppintson az attribútumok gombra az attribútumok megnyitásához.

Koordinátára ugrás¶

A keresés eszközzel lehetséges koordinátákra ugrani.

Koppintson a koordinátákra az eredményben a koordinátákra ugráshoz. WGS84 koordinátákat (hosszúság, szélesség) és aktuális vetületi koordinátákat (x,y) mindig kereshet.

Search in value list¶

It is possible to search value in a value list using the magnifying glass next to the field.

To be able to use the „completer” the widget has to be set up Autocomplet form set up.

Keresés a rétegekben¶

Elemeket kereshet a vektor rétegekben. A keresést ugyanúgy hajtjuk végre mint az összes réteg lokátor szűrő a QGIS-ben (a QGIS lokátor sorban érhető el az af előtaggal).

Keresés konfigurálása QGIS-ben¶

  1. Válasszon egy kereshető réteget a Projekt ‣ Projekt tulajdonságok ‣ Réteg azonosítás menüben.
  2. Define the display expression for the concerned layers, this will be used for searching for features. This is configured under Vector Layer Properties ‣ Display page
