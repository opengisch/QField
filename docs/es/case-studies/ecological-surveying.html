Mejorar la eficiencia en estudios ecológicos usando QField¶

Por Ryan Bell, consultor SIG, Consultora Delta-Simons Environmental

Objetivo¶

Llevar a cabo levantamientos ecológicos de manera más eficiente de lo que se había logrado anteriormente mediante técnicas tradicionales no digitales, utilizando QField como medio para la recopilación de datos de campo estandarizada y flexible.

Requerimientos del proyecto¶

Para facilitar un trabajo de campo eficaz, se requiere una consulta regular con el equipo de Ecología para garantizar que los proyectos de QField se preparen de manera adecuada. En primer lugar, se preparan mapas base que comprenden imágenes aéreas y planes de desarrollo propuestos mediante georreferenciación y construcción de pirámides en QGIS. Luego, se crean capas vectoriales de puntos, líneas y polígonos para respaldar la digitalización y la recopilación de datos asociados. Usando una variedad de widgets, los campos de atributos se personalizan para cumplir con los requisitos del equipo de Ecología. Éstos incluyen:

Estilo
lista desplegable de estilos seleccionables diseñados para realizar estudios ecológicos
Escala
lista desplegable de escalas seleccionables para la característica de punto y línea
Información
espacio para insertar una descripción de función
Imagen
ubicación para capturar y almacenar imágenes
Geometría
contiene valores de geometría predeterminados,por ejemplo: área, perímetro, longitud
Registro de tiempo
registra la hora y la fecha en que se creó / modificó por última vez una característica

Resultado¶

Delta-Simons ahorra aproximadamente el 40% del tiempo que antes consumía la recopilación de datos de campo ecológicos tradicionales. QField ha mejorado significativamente el flujo de trabajo de topografía ecológica al:

  • Eliminando la necesidad de recrear salidas GIS a partir de dibujos físicos
  • Reducir las modificaciones generales necesarias para los productos de SIG
  • Mejorar los canales de comunicación (ver Figura 1.)
  • Mejora de la precisión y calidad de los datos.
