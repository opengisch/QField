Símbolos SVG personalizados y ajustes.¶

Desde QGIS 3.6 Noosa es posible incrustar símbolos SVG en proyectos QGIS.

  1. Choose the layer which will support SVG symbology and open its properties dialog.

  2. Abrir la sección Simbología en :menuselection: Propiedades –> Simbología.

  3. En el Panel de capas de símbolos elegir Marcador simple.

  1. Change the Symbol layer type in :menuselection: Symbol layer type –> SVG marker.
  1. Scroll down the bottom panel.

  2. Clic en el lado derecho del botón de selección de archivo para abrir el menú desplegable

  1. Select Embed File and choose the SVG file in the file selection dialog.

  2. Aplicar los cambios y hacer clic en OK.
