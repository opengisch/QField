Variables globales¶

Las variables se usan en QField y QGIS para establecer dinámicamente valores basados en el archivo de proyecto actual, la capa o el sistema. Pueden usarse para aplicar estilos, establecer valores predeterminados para campos y mucho más.

Para configurar una variable específica de dispositivo, abra los ajustes y pulse en la pestaña «Variables globales».

Configuración de dos nuevas variables globales adicionales a las variables de sistema predefinidas.

En la página de variables, puede ver variables predefinidas. Para añadir nuevas variables, haga clic en la celda vacía de abajo y escriba el contenido.
