Instalación¶

Android¶

QField se puede usar con cualquier versión posterior a Android 5. Si está pensando en un nuevo dispositivo, le sugerimos ir al menos a por Android 7.

Obtenga un apk pre-construido¶

También ofrecemos un apk pre-construido para su descarga si no puede o no quiere utilizar GooglePlay. Puede descargar el apk de qfield desde la página de lanzamientos.

Windows¶

Se puede encontrar una versión beta de QField en https://download.opengis.ch/qfield/ci-builds/win/

iOS¶

Se está preparando una versión para iOS.
