Sincronización¶

A sincronización dos teus proxectos cos dispositivos dos equipos, pode facerse de varias maneiras. O plugin QFieldSync axuda a preparar e empaquetar proxectos de QGIS para QField e distribuílo entre os dispositivos. Para unha experiencia de integración fluída e para xestionar equipos recomendamos enormemente utilizar QFieldCloud.
