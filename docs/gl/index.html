QField - a solución [Q]GIS móbil¶

QField permíteche traballar eficientemente nos teus datos SIX de campo.

QField é unha interface de usuario optimizada para dispositivos Android que agocha toda a potencia de QGIS baixo o capó.

Note

Slides from FOSS4G 2016 are available.

Preparar unha vez - Utilizar en todas partes¶

  • Utiliza proxectos de QGIS (incluída a simboloxía e os widgets de edición)

  • Admite a maioría dos formatos admitidos en QGIS (ideal con PostGIS)

  • Sincronización automática de capacidades (Cable, WiFi, Nube)

  • Expande sen problemas o teu GDI

Feito para tocar - Amigo do traballo de campo¶

  • Interface liviá con poucos e grandes botóns

  • GPS céntrico

  • Funciona completamente sen conexión e en liña utilizando conexións á BD tanto en directo como gardadas na caché*

  • Modos de traballo alternos (visualización, inspección, dixitalización, medida)*

  • Ferramentas transversais (Desprazar, zoom, identificar, coordenadas GPS, barra de escala, punto de mira central con autoaxuste)

Faino medrar - Faino teu¶

  • Desenvolvemento de características baixo pago

  • Adaptable ás túas necesidades

  • Doazóns e patrocinio (qfield.opengis.ch/donate)

  • Contrátanos para desenvolvemento, consultoría e formación de QField, C++, Python, Web, QGIS, Android, ...
