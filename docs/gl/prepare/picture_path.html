Ruta de imaxe editable¶

QFieldSync dáche a posibilidade de configurar a ruta ós anexos da imaxe.

  1. Vai ó plugin QFieldSync –> Configuración do Proxecto
  2. Selecciona a táboa «Nomenclatura da Foto»
  3. Escolle a capa, o campo e configura a expresión

Utiliza as expresións para configurar a ruta dos anexos. De forma predeterminada, as imaxes gárdanse no directorio DCIM coa data e hora como nome.

Note

Precísase QGIS 3.14 ou máis recente para esta funcionalidade
