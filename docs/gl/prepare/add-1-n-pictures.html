Engade unha serie de imaxes a unha entidade¶

Podes engadir unha ou máis imaxes á entidade. Debaixo podes atopar un exemplo de como proceder.

Crear dúas táboas¶

Necesitaremos dúas táboas. Unha táboa onde as entidades se almacenan e outra cunha listaxe de imaxes.

Colmea¶

  • id (UUID)
  • geometry
  • …

Imaxes_colmea¶

  • id
  • apiary_id (UUID)
  • path (TEXT)
  • …

Relacións¶

Crear unha relación con:

  • Capa que fai referencia a colmea
  • Campo que fai referencia á id
  • Capa que fai referencia a imaxes_colmea
  • Campo que fai referencia á id_colmea
  • strength Composición

Widgets¶

colmea¶

Establecer o valor predeterminado do campo id a uuid(). Non se precisa amosar no formulario.

Establecer o widget da relación a Relación de moitos a un e engade a relación ó formulario

imaxe¶

Establecer o tipo de widget da ruta do campo a adxunto e engádeo ó formulario

Xeoetiquetado¶

Algúns dispositivos precisarán que se instale Open Camera para activar o xeoetiquetado.

Para activar o xeoetiquetado segue estes pasos:

  1. En QField, vai á configuración e asegúrate que «Utilizar Cámara nativa» está activado
  2. Instala a aplicación Open Camera no dispositivo móbil
  3. Dentro da configuración de Open Camera, asegúrate que está activado o xeoetiquetado
  4. Dentro da configuración do dispositivo, establece a aplicación predeterminada de cámara a Open Camera
  5. Completado! Utilizarase agora Open Camera para tomar fotos con QField
